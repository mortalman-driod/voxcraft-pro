# 🎙️ VoxCraft Pro

VoxCraft Pro is a high-performance, 100% free local voice-over agent designed for professional video production. It leverages Microsoft Edge's Neural TTS to generate high-fidelity voices without any API costs.

## 🚀 Key Features

- **60+ Neural Voices**: High-fidelity voices across 15+ languages.
- **Voice Studio**: Compare voices side-by-side.
- **Background Music Mixer**: Generate and mix ambient cinematic pads.
- **Video Timeline Sync**: Match your voiceover duration to your video perfectly.
- **Pro Export**: MP3, WAV, SRT subtitles, and preview MP4 videos.
- **Privacy First**: Everything runs locally in your browser/server.

## 🛠️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/voxcraft-pro.git
   cd voxcraft-pro
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   streamlit run voiceover_agent.py
   ```

## 📦 Tech Stack

- **Framework**: Streamlit
- **TTS Engine**: Edge-TTS (Microsoft Neural)
- **Audio Processing**: FFmpeg (via subprocess)
- **Visualization**: Matplotlib & Numpy
